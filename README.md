# Italian Wedding Venue Re-Evaluation

## Updated brief

You want something as visually strong as **Villa di Striano**, with **beauty and tree-filled grounds as the top priorities**, and only need a **pool suitable for a post-ceremony jump** rather than a major swim-focused weekend element.

## New scoring order

1. **Beauty / atmosphere**
2. **Trees / shaded grounds**
3. **Pool only needs to support a post-ceremony jump**
4. **General wedding practicality**
5. **Pricing / value**

## My blunt final shortlist now

### Final 5

2. **[Villa Grabau](venues/02-villa-grabau.md)**
3. **[Villa Corsini a Mezzomonte](venues/03-villa-corsini-a-mezzomonte.md)**

### Most balanced final 3

1. **[Villa di Maiano](venues/01-villa-di-maiano.md)** — best pure beauty
2. **[Villa Grabau](venues/02-villa-grabau.md)** — best beauty + trees + practicality
3. **[Tenuta Corbinaia](venues/04-tenuta-corbinaia.md)** — best beauty + reliable pool + wedding functionality

### Important caution

If the **post-ceremony jump-in is non-negotiable**, then **[Tenuta Corbinaia](venues/04-tenuta-corbinaia.md)**, **[Villa Le Fontanelle](venues/05-villa-le-fontanelle.md)**, and **[Villa Scorzi](venues/06-villa-scorzi.md)** are safer than **[Villa di Maiano](venues/01-villa-di-maiano.md)** or **[Villa Corsini a Mezzomonte](venues/03-villa-corsini-a-mezzomonte.md)**, where the pool setup should be confirmed directly.
